Quick Start (3 Steps)

  1. Install the Recipe - Visit TRMNL Plugins, find "Max Payne Quotes" in the Recipes section, and click Install
  2. Add to Playlist - Go to Playlists, edit your playlist, add "Max Payne Quotes", and select your preferred layout
  3. Enjoy! - Your TRMNL will automatically display a new quote every 24 hours

Note: If you want to customize the markup, quotes, or styling, see the Setup Guide for developers.
